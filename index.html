📌 Seu QR Code

O que é este QR Code?

Este código armazena as informações que você preencheu e podem ser lidas pela câmera de um celular.


Como usar?

Aponte a câmera para o QR Code para visualizar os dados.

Você pode imprimir e colocar em objetos pessoais como: bolsas, crachás, chaveiros.


📌 Gere o QR Code e salve ou imprima imediatamente para uso.

⚠️ Não inclua informações sensíveis ou que não possam ser compartilhadas.
